This diagram breaks down the backend work we completed for the company, an educational tutoring service and describes the intended functionality of our back-end program. This is a Python program hosted using AWS, to be called by our JavaScript frontend. The backend receives information from the frontend, the theme and problem type, and that information is passed to a function hosted on AWS lambda. That function is responsible for creating a prompt which is then sent to an OpenAI API for processing. Once processed, a problem set is generated based on the prompt and returned to the front end.
